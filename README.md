# bankAccount
## ⛳Ground Rule
- 토요일 저녁까지 pr날리기
- branch 이름은 자기 이름으로 하기
- 하다가 의논할 거 생기면 언제든 카톡하기
- 새벽카톡 허용인데 안봐도 뭐라 안하기
- 일요일 지각하면 커피사기 `이디야`
## 🖥️Developer Rule
- IDE는 인텔리제이로
- 구글 스타일 컨벤션으로 인텔리제이 포매터 설정하기
#### ✏️PR Template
```
## pr타이틀 //지워서 보내주세요

## 변경사항

## 구현사항

## 고민사항
```
#### 👿Commit Convention
```
- [FEAT]: 기능 추가
- [DOCS]: 문서 작성
- [REFACTOR]: 리펙토링
- [FIX]: 오류 수정
- [TEST]: 테스트 코드 추가
- [STYLE]: 네이밍 수정
```
